Spring

Spring翻译春天,给软件行业带来了春天前言这些东西就不说了

Spring有哪些优点呢?

  1. Spring它免费啊,我直接白嫖

  2. Spring轻量级框架,非入侵性

  3. 控制翻转IOC,面向切面AOP

  4. 对事务的支持,对框架的支持

多的不说了,就是香

总得一句话概括

Spring是一个轻量级的控制反转和面向切面的容器.

Spring的模块结构

简短直说

  1. 核心容器:spring的基础功能,核心组件就是BeanFactory,他是工厂模式(设计模式)的实现,就这个bean工厂使用的就是IOC将应用程序的配置和依赖性规范与实际的应用程序代码分开.简称高内聚低耦合

  2. SpringAOP:可以通过配置管理特性, 简称面向切面 直接集成到Spring框架中

  3. SpringDAO: 自行百度 我自己都懒得看

  4. SpringORM: 自行百度 我自己都懒得看

  5. SpringWeb框架: 自行百度 我自己都懒得看

  6. SpringMVC框架: MVC框架是一个全功能的web应用程序的MVC实现 其中容纳大量的视图技术,其中包含咱常见的jsp,pol

M Model 模组 表示业务规则

V View 视图 表示用户看到并与之交互的界面

C Controller 控制器 表示接受用户输入并调用模组和视图去完成用户的需求
